# Java Solutions for Advent of Code

This project contains solutions written in Java.

## Setup

Gradle is used for building and running.
All commands must be run in this directory.

## Running the Solutions

Run `gradle run --args="<year> <day>"` to compute the solution for the given year and day.
For example, `gradle run --args="2023 1"` runs Advent of Code 2021&#8209;12&#8209;01.

## Testing

Run `gradle test` to run all tests.
Use `gradle test --tests SomeTestClass` or `gradle test --tests SomeTestClass.someSpecificMethod` to execute a specific class ot method.
